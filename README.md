# BoringKiller
``` c++
***********************************************************************************************************************
功能介绍：
  全局捕获26字母按键，每按下一个字母的时候，会播放一个音符。
  在bin目录下面运行WanderCat.exe即可启用音乐键盘功能
  源码可在windows下进行编译，且需要链接libwinmm.a
***********************************************************************************************************************
文件结构 ：
|
|-----README.md
|-----src
|      |----main.cpp           
|------bin
|      |-----WanderCat.exe     main.cpp生成的可执行文件
|      |-----sound/            音乐资源文件(来自https://github.com/itmojun/my_piano)      
|      |-----琴谱.txt          琴谱(来自https://github.com/itmojun/my_piano)
|
***********************************************************************************************************************
源码参考
按键捕获模块：https://github.com/Harkphoenix/KeyBoardWriter/
播放音频资源模块: https://github.com/itmojun/my_piano
***********************************************************************************************************************
```
